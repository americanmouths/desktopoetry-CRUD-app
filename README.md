# Desktopoetry
A Sinatra web application designed to provide ability to create, read, update & destroy all things poetry.

# Usage
To use this app:
- Clone
- In terminal run bundle install
- Run rake db:migrate
- Run shotgun
- Navigate to the local host displayed by shotgun
